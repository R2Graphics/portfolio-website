The Cookie Jar

This was a fun branding project. A new bakery got a playful logo that used smart typography techniques, an advertisement that drew the attention of potential buyers, a price list that was created to be easy to read and very informative, and much more. This project was done over a few weeks using Adobe Suite- Photoshop, Illustrator, and InDesign.
